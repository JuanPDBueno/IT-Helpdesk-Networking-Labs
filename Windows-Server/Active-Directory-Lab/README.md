# Active Directory Helpdesk Lab

## Objective
Simulate a real-world helpdesk environment using Windows Server and Active Directory.

## Environment
- Windows Server 2022 (Domain Controller)
- Windows 10 clients
- VirtualBox

## Key Tasks
- Installed and configured Active Directory Domain Services
- Configured DNS and DHCP
- Created users and organizational units
- Joined client machines to the domain
- Managed shared folders and permissions

## Troubleshooting Scenarios
- User account lockout
- Password reset
- Client unable to join domain due to DNS issue

## Outcome
Successfully deployed a functional domain environment and resolved common helpdesk issues.
